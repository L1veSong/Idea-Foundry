一个想法 → 一键锻造 → 可交付成果

---

## 版本

当前: **v9.0.0 — 角色纪元**

### v9 核心：角色声明制

废除数字优先级竞争。调度权来自 `role: orchestrator, stage: dispatch`。其他 Skill 声明 `role+stage` 即可协作。

### 功能亮点

- 🏗️ 角色声明制 — 数字竞争→角色协作
- ⬅️ 上一步回退 — 所有菜单可回退重选
- ⛔ 检查点铁则 — 结构化执行规则表
- 🆕 自进化 — 陌生领域自动生成标签+插件
- 📊 自适应仲裁 — 历史成功率动态调优
- 🧪 自检命令 — `foundry test --self`

## 安装

```bash
bash scripts/install.sh          # macOS / Linux
powershell scripts/install.ps1   # Windows
```

---

## 使用

```
「帮我做一个摄影作品集网站」   # 自动触发
「用极致模式帮我设计这个API」  # 手动指定
「foundry hold」              # 暂停
「foundry resume」            # 恢复
「foundry version」           # 版本
「foundry test --self」       # 自检
```
